The project involves real-time sectroscope analysis using a live video feed. The program captures video frames from a webcam and extracts spectral information, generating graphs of reflectance and transmittance. This tool can be used for various applications, including material analysis, chemical testing, and educational purposes. I used python and OpenCV to develop this project.
